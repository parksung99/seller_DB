# 셀러 크롤링 전략 정리

## 최종 목표

셀러 크롤링의 최종 목표는 흩어져 있는 셀러 데이터를 수집하고, 이를 브랜드가 의사결정에 활용할 수 있는 자산으로 만드는 것이다.

단순히 셀러 리스트를 모으는 것이 아니라, 각 셀러의 채널 활동, 판매 방식, 콘텐츠 특성, 고객 반응, 강점 등을 데이터화하여 브랜드에게 접근 가능한 형태로 전달하는 것이 핵심이다.

## 핵심 방향

### 1. 데이터를 자산화한다

크롤링으로 수집한 셀러 정보를 정리하여 브랜드에게 전달 가능한 데이터 자산으로 만든다.

수집 대상 데이터 예시는 다음과 같다.

- 셀러명
- 운영 채널
- 채널 URL 또는 계정 정보
- 주요 판매 카테고리
- 사용 해시태그
- 콘텐츠 업로드 빈도
- 팔로워 또는 커뮤니티 규모
- 댓글, 좋아요, 반응 수
- 판매 방식
- 공동구매 여부
- 브랜드 협업 가능성
- DM 발송 가능 여부

이 데이터는 이후 브랜드 영업, 셀러 제안, 캠페인 매칭, 플랫폼 입점 제안에 활용한다.

### 2. 셀러의 장점을 데이터로 만든다

크롤링을 통해 발굴한 셀러가 우리 플랫폼과 함께할 가능성이 있다면, 그 셀러만의 장점을 데이터로 정리해 브랜드에게 전달한다.

예를 들어 다음과 같은 방식으로 셀러의 강점을 표현할 수 있다.

- 특정 카테고리에 강한 셀러
- 고객 반응률이 높은 셀러
- 공동구매 경험이 많은 셀러
- 특정 지역 또는 관심사 기반 커뮤니티를 보유한 셀러
- 콘텐츠 제작력이 좋은 셀러
- 브랜드 제품을 자연스럽게 소개할 수 있는 셀러

즉, 브랜드에게는 "이 셀러와 왜 협업해야 하는가"를 숫자와 근거로 설명할 수 있어야 한다.

### 3. 채널별 크롤링 방법을 다르게 설계한다

카카오톡, 밴드, 네이버, 인스타그램은 구조와 접근 방식이 다르기 때문에 채널별로 별도의 크롤링 전략이 필요하다.

#### 인스타그램

- 해시태그 기반 셀러 탐색
- 프로필 소개 문구 분석
- 게시물 업로드 빈도 확인
- 좋아요, 댓글 등 반응 데이터 수집
- 공동구매, 마켓, 오픈채팅, DM 문의 키워드 확인
- 1차 DM 발송 대상 리스트 구축에 가장 우선적으로 활용

#### 네이버

- 블로그, 카페, 스마트스토어, 검색 결과 기반 탐색
- 제품 리뷰, 공동구매 글, 판매 키워드 수집
- 셀러의 검색 노출력과 콘텐츠 신뢰도 확인
- 브랜드 협업 가능성이 있는 운영자 또는 판매자 발굴

#### 밴드

- 폐쇄형 또는 반폐쇄형 커뮤니티 특성을 고려
- 공동구매, 지역 기반 판매, 관심사 기반 판매 그룹 탐색
- 멤버 수, 게시글 빈도, 판매글 반응 확인
- 커뮤니티 영향력이 있는 운영자 중심으로 정리

#### 카카오톡

- 오픈채팅방, 채널, 커뮤니티 기반 접근
- 직접 크롤링 난이도가 높기 때문에 수집 가능한 공개 정보 중심으로 설계
- 초대 링크, 채널명, 참여자 규모, 판매 방식 등을 우선 정리
- 이후 수동 검수 또는 반자동 수집 방식과 병행

## 1차 목표

1차 목표는 해시태그 기준으로 셀러들을 최대한 많이 모아 DM 발송 리스트를 만드는 것이다.

우선순위는 다음과 같다.

1. 인스타그램 해시태그 기반 셀러 수집
2. 판매 또는 공동구매 관련 키워드로 계정 필터링
3. DM 발송 가능 계정 선별
4. 셀러 기본 정보 정리
5. 브랜드 협업 가능성이 높은 셀러 우선순위화

## 1차 DM 리스트에 포함할 필드

DM 발송 리스트는 최소한 아래 필드를 포함한다.

| 필드 | 설명 |
| --- | --- |
| seller_name | 셀러명 또는 계정명 |
| channel | 수집 채널 |
| url | 계정 또는 채널 URL |
| hashtag | 발견된 주요 해시태그 |
| category | 판매 카테고리 |
| follower_count | 팔로워 또는 멤버 수 |
| engagement_signal | 좋아요, 댓글, 조회수 등 반응 지표 |
| selling_signal | 판매, 공동구매, 마켓 운영 여부 |
| dm_available | DM 발송 가능 여부 |
| priority | 연락 우선순위 |
| notes | 추가 메모 |

## 실행 로드맵

### 1단계: 해시태그 기준 수집

인스타그램을 중심으로 판매자 가능성이 높은 해시태그를 정리하고, 해당 해시태그에서 셀러 후보 계정을 수집한다.

예시 해시태그:

- 공동구매
- 공구
- 마켓
- 인스타마켓
- 육아템공구
- 뷰티공구
- 패션마켓
- 다이어트공구
- 살림템
- 주부마켓

### 2단계: 셀러 후보 필터링

수집한 계정 중 실제 판매 활동이 있는 계정을 선별한다.

판단 기준:

- 프로필에 판매, 마켓, 공구, DM 문의 등 키워드가 있는가
- 최근 게시물에 상품 소개 또는 판매 안내가 있는가
- 댓글과 좋아요 반응이 꾸준한가
- 외부 링크, 오픈채팅, 스마트스토어, 블로그 링크가 있는가

### 3단계: DM 발송 리스트 구축

필터링된 셀러 후보를 DM 발송 가능한 리스트로 정리한다.

리스트는 단순 연락처 목록이 아니라, 왜 연락해야 하는지 판단할 수 있는 근거를 함께 포함해야 한다.

### 4단계: 브랜드 전달용 데이터 구조화

플랫폼과 함께할 가능성이 있는 셀러는 별도로 정리하여 브랜드에게 전달 가능한 형태로 가공한다.

브랜드 전달용 요약 예시:

- 이 셀러가 강한 카테고리
- 주요 고객층
- 콘텐츠 스타일
- 예상 협업 적합도
- 과거 판매 또는 공동구매 신호
- 브랜드 제품과의 매칭 가능성

## 최종 산출물

초기 단계에서 만들어야 할 산출물은 다음과 같다.

- 해시태그별 셀러 후보 리스트
- DM 발송 대상 리스트
- 채널별 크롤링 전략 문서
- 셀러별 강점 요약 데이터
- 브랜드 전달용 셀러 리포트 초안

## 핵심 문장

우리가 만드는 것은 단순한 셀러 크롤링 리스트가 아니라, 브랜드가 바로 협업 판단에 사용할 수 있는 셀러 데이터 자산이다.
